# pm

直進優先型ウォーターフォールPMアプリです。

Googleカレンダーから制作予定を取り込み、タスク、進捗、遅延対応、修正リスト、ワークスペース管理を行うためのアプリです。

## 現在の主な課題

- 初回ログイン時、ワークスペース未作成なのに取得処理へ進んでしまう
- ワークスペース作成に時間がかかる
- 作成中UIが分かりづらい
- ログイン・アカウント切替・ワークスペース作成周辺の異常系を整理中

## 開発方針

- mainブランチへ直接大きな変更を入れない
- 1回の修正では1テーマだけ扱う
- 実装前に要件・フィードバックレポートを作成する
- Claude Codeには実装を担当させる
- ChatGPTでは要件整理・レビュー・指示文作成を行う
