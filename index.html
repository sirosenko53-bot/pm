🔧 初期設定

Firebaseの設定情報を貼り付けてください。
設定はこのブラウザにのみ保存されます。

1

console.firebase.google.com でプロジェクトを作成
→ 「ウェブアプリを追加」→ アプリ名を入力 → 「Firebase SDK の追加」

2

表示された firebaseConfig の { } の中身をコピー
（apiKey, authDomain, projectId ... が含まれるJSON）

3

以下に貼り付けて「保存して開始」

設定の形式が正しくありません。JSONを確認してください。

詳しい設定手順は付属の README.md を参照してください
