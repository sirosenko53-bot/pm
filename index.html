🔧 初期設定

Firebase設定とGoogle OAuth Client IDを入力してください。

1

console.firebase.google.com でプロジェクト作成 → ウェブアプリ追加 → firebaseConfig の { }の中身をコピー

2

console.cloud.google.com → 「APIとサービス」→「認証情報」→ OAuthクライアントID（ウェブ） の クライアントID をコピー
（例: 123456.apps.googleusercontent.com）

JSONの形式が正しくありません。

詳細はREADME.mdを参照してください
