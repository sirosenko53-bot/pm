# AGENTS.md

このリポジトリでは、Codexはコード実装のみを担当する。

要件定義、UI設計、仕様判断、優先度判断、レビュー、異常系整理はChatGPT側で行う。
Codexは、渡された実装指示に従ってコードを変更する。

## 作業ルール

- 指示された範囲だけを実装する。
- 指示外の仕様変更、大規模リファクタリング、機能追加はしない。
- 既存機能を削除する場合は理由を説明する。
- UI文言は日本語で統一する。
- ログイン、ワークスペース、Google Calendar同期、削除処理は慎重に扱う。
- 似た関数を増やして旧処理を放置しない。
- 作業後は変更点、確認手順、未確認点だけを報告する。
- 新しくブランチを作る際は名前を短く、バージョン名にする

## 禁止事項

- 要件を勝手に再設計しない。
- UI/UX方針を勝手に変えない。
- 指示外の機能追加をしない。
- mainへ直接反映しない。
- 動作確認なしに完了報告しない。
- Googleカレンダー同期、削除、ログイン処理を同時に大改修しない。

## 報告形式

1. 変更したファイル
2. 変更した関数・画面
3. 実装内容
4. 動作確認手順
5. 未確認点

## Review guidelines

- 指示範囲外のファイル変更がないか確認する。
- 危険操作に確認UIがあるか確認する。
- ログイン、ワークスペース、Google Calendar同期、削除処理に副作用がないか確認する。
- 日本語UI文言が不自然に英語化されていないか確認する。
